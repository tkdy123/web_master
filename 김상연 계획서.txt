1) 인덱스 페이지 (index.html)
	1. 샘플데이터를 로컬스토리지에 등록

2) 로그인 페이지 (login.html)
                1. 아이디가 없을 시 회원가입
	2. 아이디와 비밀번호 입력 후 로그인
	3. 로그인 성공 시 웰컴 페이지로 이동
3) 회원가입 페이지(signup.html)
               1. 회원정보 입력 후 가입
                
4) 메인 페이지 (welcome.html)
	1. 사이트에서 제공하는 메뉴들(한줄평 입력페이지, 내가 쓴 한줄평 페이지 ,리뷰확인 페이지) 출력

5) 영화선택 페이지 (movie.html)
	1. 영화 제목, 사진, 줄거리 출력
	2. 선택한 영화 리뷰쓰기
                3. 로컬스토리지에 등록
6) 내가 쓴 한줄평 페이지 (myreview.html)
                1. 로그인 한 아이디로 작성한 한줄평 출력, 삭제기능

7) 리뷰확인 페이지 (review.html)
	1. 모든 이용자의 한줄평 출력, 추천기능
