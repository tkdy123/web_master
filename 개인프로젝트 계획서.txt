프로젝트 계획서 샘플
< 주제 : 상품 & 장바구니 >
1) 인덱스 페이지 (index.html)
	1. 샘플데이터를 로컬스토리지에 등록
	2. 로그인 페이지로 이동

2) 로그인 페이지 (login.html)
	1. 아이디와 비밀번호 입력 후 로그인
	2. 로그인 성공 시 메인 페이지로 이동

3) 메인 페이지 (welcome.html)
	1. 사이트에서 제공하는 메뉴들 출력

4) 상품목록조회페이지 (product.html)
	1.상품정보 전체 출력
	2. 선택한 상품을 장바구니에 담기

5) 장바구니 페이지 (cart.html)
	1. 장바구니 목록 출력
	2. 각 상품의 구매수량 업/다운 후 구매금액 연동
	3. 전체 구매금액 합계 출력
	4. 선택 상품 삭제 및 전체 삭제
